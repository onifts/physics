- Entropy 
  - 무질서 
  - 엔트로피가 높다 = 무질서가 높다.
  - 자연 물질이 변형되어 다시 돌아오지 못하는 상태를 엔트로피 증가의 법칙이라고 한다.
  - 에너지의 사용은 결국 사용가능한 에너지가 손실되는 결과를 가져온다.
  - 열은 뜨거운 -> 차가운 곳으로 이동한다. (에너지의 이동) 
  - 열의 전달 : 뜨거운 곳에 엔트로피가 높은 전자가 차가운 곳에 엔트로피가 낮은 전자와 충돌하여 엔트로피를 높이면서 열이 전달된다.
